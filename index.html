About these conversions & data sources
Settings are approximated via linear interpolation based on each grinder's published micron range. Timemore C5 ESP Pro: 0 – 1,250 μm across 150 total steps (0.0.0 → 3.0.0), ≈ 8.3 μm per step. Timemore C5 Pro: 0 – 950 μm across dial settings 1.0 → 16.0. Timemore S3: 330 – 1,110 μm across clicks from zero 0.0 → 9.0. Comandante C40 MK4: 0 – 1,090 μm across 40 clicks, ≈ 27.3 μm per click. Baratza Encore: 250 – 1,200 μm across 40 steps, ≈ 23.75 μm per step.

The Timemore notation R.N.T reads as: R = full rotations (0–3), N = number mark (0–9), T = tick between marks (0–4). Each full rotation contains 50 steps (10 numbers × 5 ticks).

Real-world grind size varies with bean density, roast level, humidity, and burr condition. Use these values as starting points and dial in to taste. Brew method ranges sourced from Honest Coffee Guide and verified against Onyx Coffee Lab.
